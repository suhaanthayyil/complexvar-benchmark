# Novelty and Scope Memo

## Positioning

This repository should not claim without qualification that it is the first
graph model for interaction-disrupting missense variants. Close prior work
exists by 2025 and 2026.

## Safe Claim

The benchmark should be framed as:

> A rigorously controlled, reproducible benchmark evaluating whether
> protein-complex structural context improves prediction of
> interaction-disrupting missense variants.

## Narrow Contributions

- matched sequence-only, monomer-structure, and complex-structure comparison
- strong non-GNN structural baselines
- explicit interface-only evaluation
- family-group leakage control
- pathway and gene-class interpretation on high-confidence human complexes
