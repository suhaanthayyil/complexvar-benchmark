# Licensing and Redistribution Memo

## Code

- Repository code license: MIT

## Data

### Burke complex and monomer model release

- Verified release license: `CC BY 4.0`
- Safe to script download and redistribute attribution-bearing derived manifests

### ClinVar

- Public NCBI distribution
- The repository ships scripts and manifests, not full raw mirrors

### SKEMPI 2.0

- Portal displays a Creative Commons badge in site footer material
- The repository will still prefer scripted download plus citation instead of
  vendoring the full raw dataset

### IntAct / IMEx

- IntAct site routes to EMBL-EBI terms of use
- Mutation exports are script-downloaded, and the repo stores only compact
  derived artifacts by default

### FoldX / Rosetta

- Not redistributable as core dependencies in this repository
- Any use is limited to optional subset audits outside the core claim

## Repository Policy

- Track scripts, manifests, checksums, label audits, and small derived assets
- Do not commit raw bulk dataset mirrors by default
- Every manuscript claim must link to code, stored result, or citation
