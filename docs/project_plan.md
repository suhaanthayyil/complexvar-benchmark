# Project Plan

ComplexVar Benchmark is organized into eight reproducible phases:

1. Verify data sources, licensing, access routes, and scope.
2. Normalize identifiers and variant labels into deterministic manifests.
3. Build reusable complex and monomer residue-graph caches.
4. Construct leakage-aware family-group splits.
5. Train sequence, structural, and graph models.
6. Evaluate with interface-focused metrics and grouped bootstraps.
7. Run biological interpretation and failure-mode audits.
8. Package manuscript assets, figures, tables, and final audit outputs.

Each phase is driven by scripts under `scripts/` and mirrored in the Snakemake
workflow.
